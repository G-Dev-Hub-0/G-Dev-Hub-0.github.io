MikOS开源以及使用协议

2023.8.28第一版

一只野生的小肝同学 作

一只悠然自得 审核

MikOS (以下简称本系统) 由 __G Dev Hub 工作室__（以下简称本工作室）MikOS开发组开发，自2.0版对本系统进行开源，使用时请务必遵守本协议

使用者

若您是使用者，请遵守本章规则

1.本系统为基于Python Tkinter 开发的类操作系统的GUI，本系统没有任何收集用户信息的功能，所有用户运行时产生的文件以及隐私文件全部将保存到用户所使用的计算机上，不会上传到任何服务器上（本工作室无法保证第三方应用的程序不会上传文件，本条协议仅限于本工作室开发的系统主程序）

2.您使用本系统后所产生的的任何经济损失以及人身伤害本工作室概不负责

3.若您有任何疑问，请通过邮箱 G_Studio_0@outlook.com 联系我们

4.不得私自将本系统用于商业用途，一经发现，本工作室将追究其责任

5.本系统的应用没有任何如收费功能、广告等盈利类功能，如果您发现了这类应用，请通过邮箱 G_Studio_0@outlook.com 联系我们，格式如下：版本+应用名称+应用版本+盈利的功能。感谢您的配合!

开发者

若您是开发者，请遵守本章规则

1.本系统基于Python开发，若您想为本系统开发第三应用，请使用Python语言，若您使用其他语言/第三方库，请务必保证向用户说明本应用所使用的语言以及需要安装的运行库 注意: 若您使用其他语言，请考虑您所使用的语言的兼容性以及在本系统上的稳定性

2.所有第三方应用请务必保存在 MikOS/xxx/app/(您的应用程序名) 文件夹中

3.文件命名规则如下(XXX代表您的程序名)：

文件类型 命名方式
主程序 (XXX_)Main
其他文件 XXX 文件名
协议 Licence (可选)
程序说明 Readme

4.若您的程序要上传用户文件，请务必在程序说明中标注所需上传的文件

6.请在Readme文件中说明您开发的应用的用途、所需的运行库以及开发语言

6.所有程序务必在系统主程序中使用
os.startfile(‘’)

命令启动您开发的应用

9.不得在应用中加入任何如广告、收费功能的盈利项目，您可以考虑加入非盈利性的广告，但请务必注明，并向本工作室申请，否则我们将追究其责任

8.开发完成后，请务必通过邮箱 G_Studio_0@outlook.com 联系我们以审核您的程序

9.联系我们时，您的程序请务必打包成压缩文件（如 *.rar, *.zip , *.7z 等），以方便我们审核，谢谢！

10.若您想为本系统的系统文件做出更改或者想加入我们的开发，请通过邮箱 G_Studio_0@outlook.com 联系我们，以便加入G Studio。未经授权，不得修改本系统的根文件，否则本工作室将追究其责任

11.不得私自将本系统用于商业用途，一经发现，本工作室将追究其责任
